Using the Public Cloud to Scale Your Capabilities

Dublin R User Group

 Tue 24 Nov 2015

 tags: r presentations public cloud rstats big data

I had the pleasure of presenting to the Dublin R User Group on 24 Nov 2015 on ‘Using the Public Cloud to Scale Your Capabilities’. My talk source and examples are in a Github repo. The talk was the third in my series on R and machine learning to the group. It was the final in the series and covered more advanced topics such as how to scale your machine learning using the Public Cloud (in this talk Amazon’s Web Services such as Elastic Computing)
