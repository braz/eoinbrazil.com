Omnigraffle, Climacons and Font Awesome

Stencils for two useful fonts

 Wed 30 Oct 2013

 tags: omnigraffle fonts stencils design

I recently been doing a few sketches using Omnigraffle and realised that I had to create my first stencils to support the graphical prototyping I’d been working on. The first font is the excellent Font Awesome and it already had a great stencil from Pat Cheung that I merely needed to update from his version to include the latest release of the font. The second font was a really nice climate icon font, Climacons that did not have any stencil.

I really took some inspiration from Pat’s work and have sent a pull request back to his Github so hopefully the stencil will be updated. I really think that Dave Gandy and all the Font Awesome contributors have really created something so useful for wire framing.

Font Awesome Stencil

Firstly, down load the system font file from the project’s website and then install this font on your computer. You can then download the stencil from my (Github project)[https://github.com/braz/Font-Awesome-OmniGraffle-Stencil], double click the stencil file to install it. The links for these are:

  • Font Awesome: http://fontawesome.io
  • Font Awesome 4.0.1 OmniGraffle Stencil github: https://github.com/braz/Font-Awesome-OmniGraffle-Stencil

Climacons Stencil

Firstly, down load the system font file from the project’s website or the Github repository and then install this font on your computer. You can then download the stencil from my (Github project)[https://github.com/braz/climacons-omnigraffle-stencil]], double click the stencil file to install it. The links for these are:

  • Climacons font: http://adamwhitcroft.com/climacons/font/
  • Climacons github: https://github.com/AdamWhitcroft/Climacons
  • Climacons Omnigraffle Stencil github: https://github.com/braz/climacons-omnigraffle-stencil

Hope you find these of use and feel free to submit pull requests as I intend to try and keep these updated.
